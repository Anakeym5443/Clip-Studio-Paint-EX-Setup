## 🖌️ Core features

This software focuses on production speed and artistic control. Explore these tools to improve your workflow.

*   Brush Engine: The engine creates unique textures. Customize settings to mimic pencils, ink pens, or paintbrushes. Pressure sensitivity ensures smooth line weight.
*   Vector Layers: Use vector layers to edit line art after you draw. Control individual points or scale shapes without losing image quality.
*   3D Reference Models: Load body models to test angles and light. Pose the models to guide your sketch. This saves time on anatomical study.
*   Manga Tools: Access panels, speech bubbles, and tone patterns. These features automate repetitive parts of comic production.
*   Animation Suite: Create frame-by-frame animations. This includes a timeline, light table functionality, and onion skinning.

## 💡 Troubleshooting common issues

If the software fails to start, check the following items.

*   Compatibility mode: Right-click the shortcut and select properties. Select the compatibility tab. Pick Windows 8 or 10 compatibility mode.
*   Administrative access: Run the setup file with administrative rights. Right-click the file and select run as administrator.
*   Antivirus interference: Some security software blocks new programs. Check your security settings if the installer refuses to open.
*   Graphics drivers: Update your display drivers through your GPU manufacturer website. Stale drivers cause issues with hardware acceleration.
